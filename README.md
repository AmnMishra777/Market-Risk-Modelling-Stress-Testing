Market Risk Modelling and Stress Testing of an HDFC Focused Fund Portfolio using Python, VaR, Expected Shortfall, CAPM, GARCH, Copulas and Stress Testing techniques.


# Market Risk Modelling & Stress Testing of HDFC Focused Fund Portfolio

## Overview

This project evaluates the market risk characteristics of a 28-stock NSE-listed equity portfolio using Python-based quantitative techniques.

## Key Techniques

- Portfolio Beta
- Sharpe Ratio
- Historical VaR
- Parametric VaR
- Monte Carlo VaR
- Expected Shortfall
- Copula-Based VaR
- CAPM
- GARCH (1,1)
- Stress Testing

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- SciPy
- Statsmodels
- ARCH

## Key Findings

- Estimated downside risk under normal and stressed market conditions.
- Evaluated portfolio sensitivity to market movements.
- Compared multiple VaR methodologies.
- Assessed volatility clustering using GARCH models.
